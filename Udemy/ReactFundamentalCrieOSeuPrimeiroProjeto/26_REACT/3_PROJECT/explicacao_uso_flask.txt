Por que o servidor flask precisa ser usado neste projeto no. 3?

O servidor flask serve para resolver o problema do servidor json-server, quando se vai atualizar ou remover uma instância de TODO (tarefa a fazer, do inglês: "to do") do arquivo json, que serve como banco de dados no servidor.
